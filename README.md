# Moodle 5.1 Software Design Project

## Overview
This repository contains all design artefacts for the **Moodle 5.1 Software Design Project** completed as part of the Software Design course. The project focuses on analyzing, modeling, and designing an open-source Learning Management System (LMS) using professional software engineering principles.

The repository includes:
- Requirement modelling
- Architectural design
- Behavioural and structural diagrams
- UI/UX mockups
- Design evaluation and documentation
